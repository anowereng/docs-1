# Agile

## SCRUM

Sprint is a 1 to 3 weeks timebox several incremented releases.

### Roles

There are 3 roles in SCRUM:

- Product owner
- Scrum master
- Team

### Artifacts

There are 3 artifacts:

- Product backlog
- Spring backlog
- Burndown chart

### Ceremonies

There are 3 ceremonies:

- Sprint planning
- Daily Scrum
- Sprint review
