# Vim

Command | Explanation
-|-
vim | Start Vim
:help \<key\> | Show help explaining what \<key\> does
:w | Write changes
:w name.txt | Write changes to file
:q | Quit Vim
:q! | Quit without saving

## Navigation

Command | Explanation
-|-
hjkl | Keys for moving in command mode
w | Go to the word end
b | Go to the word begining
$ | Go to end of line
^ | Go to begining of line
